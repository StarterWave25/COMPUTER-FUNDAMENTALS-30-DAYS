# **Central Processing Unit**

## **Why was CPU invented?**
- ### Humans want to automate some tasks like performing calculations.
- ### They want a machine that can understand instructions and performs a task.
- ### That machine is called the **CPU**.

## **What problem exists before the CPU?**
- ### Humans has to do calculations which are slow & error-prone.
- ### There are calculators for different operations like addition, subtraction, multiplication, etc. No calculator does all operations.
- ### If a computer wants to perform another type of operation then the cables need to be rewired the 100's of cables. 
- ### And the instructions & data of a program cannot be stored in the memory. As stored program concept is not exists before. So, they have to be given for every program.

## **What would happen if the CPU did not exist?**
- ### As every OS, App, Operation, Game has to become instructions that has to be executed but if CPU doesn't exists then they don't exists at all.
- ### Calculations needs to be performed with different calculators for different operation.
- ### Laptops, Mobiles, Smart tvs, game console don't exists.
- ### Our daily activties rely everything on papers.

## **A Real world analogy of CPU**
- ### Imagine a worker building a house by following these steps:
        Pick Brick -> Place Brick -> Apply Cement -> Pick Brick -> Place Brick -> Apply Cement -> ...
- ### They are all thousands of steps but after completing them you will see a big beautiful house.
- ### Same for instagram, it is a beautiful app but behind the scenes the CPU executes billions of instructions to build it.

## **Simple Example to understand the flow of the CPU**
- ### let total = 5 + 10;
- ### When this is program is interpreted it is converted into Binary instructions and stored in RAM.
- ### The instructions looks like in Assembly level:
        LOAD R1 5 -> Operands are stored in CPU Registers
        LOAD R2 10
        ADD R3 R1 R2 -> Result is stored in R3 Register
        MOV R3 total(2404) -> The R3 is moved to the address of total variable in RAM.
